Some operations take noticeable time -- joining a mesh, configuring WireGuard, downloading binaries. The CLI displays animated progress indicators so the operator knows something is happening and the process has not hung.

## Spinner

A spinner is shown for single-step operations that do not have a predictable duration:

```
  ⠸  Joining mesh...
```

The spinner animates in place with a braille dot pattern. When the operation completes, the spinner is replaced by a final status message on the same line:

```
  ✓  Joined mesh successfully.
```

On failure:

```
  ✖  Failed to join mesh.
```

The checkmark is green and the cross is red when color is enabled.

### Detail output

Some operations append a right-aligned detail to the success message, providing extra context without a separate line:

```
  ✓  Scheduled on hypervisor                    fsn1
```

## Multi-step progress

For operations with multiple sequential phases, a combined spinner and progress bar shows both the current step and overall progress:

```
  ⠙  Configuring WireGuard          ━━━━━━━╸──────────── 3/7
```

As each step completes, the bar advances and the message updates to describe the next step. When all steps finish, the entire line is replaced with a single success message:

```
  ✓  Hypervisor configured.
```

On failure at any step:

```
  ✖  Failed at: Configuring WireGuard
```

## Download progress

For operations that transfer data (downloading binaries, syncing state), a progress bar with byte counts is displayed:

```
  ↓  Downloading nauka-agent  ━━━━━━━━━╸──────── 12.4 MiB/45.0 MiB
```

When the download completes:

```
  ✓  Downloaded nauka-agent
```

## Non-TTY behavior

When output is not going to a terminal (piped to a file, running in CI), spinners and animated progress bars are suppressed. Instead, the CLI prints plain-text status lines:

```
  Configuring WireGuard...
  Setting up peering...
  ✓  Hypervisor configured.
```

No animation, no ANSI escape codes. This keeps log files readable and avoids polluting piped output with control characters.
