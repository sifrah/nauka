nauka_compute::vm

Module provision 

Source
Expand description

VM provisioning — TAP interface creation.

Each VM gets a TAP interface attached to its VPC bridge:

  • nkt-{hash} — TAP interface (will be passed to Cloud Hypervisor)

The hash is derived from the VM ID.
