Expand description

The bootstrap-state SurrealQL schema, embedded at compile time.

Defined in layers/state/schemas/bootstrap.surql (P1.6, sifrah/nauka#196). EmbeddedDb::open applies this on every open. The schema is fully idempotent thanks to IF NOT EXISTS on every DEFINE statement, so re-applying against an already-initialised database is a no-op.
