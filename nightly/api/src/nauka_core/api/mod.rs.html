1//! API layer — auto-generates REST routes from ResourceDef.
2//!
3//! Layers export pure handlers. This module wraps them in HTTP:
4//! - Routing: ResourceDef → axum routes (with scoped paths)
5//! - Request parsing: JSON body + path params → OperationRequest
6//! - Response rendering: OperationResponse → JSON HTTP response
7//! - Error handling: NaukaError → HTTP status + JSON error body
8//! - Middleware: request ID, version header, rate limiting, tracing
9//! - Pagination: standard query params on list endpoints
10//! - OpenAPI: auto-generated spec from ResourceDef
11//! - Graceful shutdown: SIGTERM/SIGINT handling
12//!
13//! ```no_run
14//! use nauka_core::api::{ApiServer, ApiConfig};
15//!
16//! # async fn example() {
17//! let config = ApiConfig::default();
18//! let server = ApiServer::new(config, vec![], vec![]);
19//! // server.run().await;
20//! # }
21//! ```
