Expand description

PD reconciler — removes zombie PD members that stay unhealthy for >5 minutes.

PD members from failed joins can linger as unhealthy phantoms, breaking quorum or preventing new joins. Previously these were only cleaned during the next hypervisor join. This reconciler catches them continuously.

Runs as a pre-flight step (before TiKV connect) because a zombie member can prevent the TiKV client from connecting at all.
