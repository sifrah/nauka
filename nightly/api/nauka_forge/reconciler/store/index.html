TiKV store reconciler — offlines stores whose fabric peer is unreachable.

When a fabric peer has been unreachable for more than 60 seconds, the TiKV store running on that peer is moved offline via the PD API. This prevents stuck regions caused by PD waiting for a store that will never return.
