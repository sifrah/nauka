Subnet persistence on the SurrealDB-backed cluster store.

P2.12 (sifrah/nauka#216) migrated this module from the legacy raw-KV cluster path to the native SurrealDB SDK on top of EmbeddedDb. Every method reaches the SDK via self.db.client() and writes/reads against the SCHEMAFULL subnet table defined in layers/network/schemas/subnet.surql.

The legacy sub / sub-idx / _reg_v2 sidecar namespaces are gone: the schema’s composite unique index on (vpc, name) plus the secondary uniqueness on (vpc, cidr) enforce every invariant the pre-P2.12 store checked manually.
