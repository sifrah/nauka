Module announce 

Source
Expand description

Peer announcement — broadcasts new peers to all existing mesh members.

When a new node joins, the accepting node sends a PeerAnnounce message to every existing peer so they can add the new node to their WireGuard config.

Protocol: TCP connect → send PeerAnnounce → close. Same length-prefixed JSON as the peering protocol.

Announcements are best-effort: if a peer is unreachable, we skip it and rely on the next health check or manual sync.
