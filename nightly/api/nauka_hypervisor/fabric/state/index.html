Fabric state — persisted mesh configuration.

The fabric state is the complete snapshot of a node’s mesh membership: mesh identity, node identity, secret, and list of peers.

Persisted in the SurrealKV-backed EmbeddedDb (P1.10, sifrah/nauka#200) as a single JSON blob at fabric:state. Read/written via the async save/load/delete/exists API. P1.11 (sifrah/nauka#201) removed the legacy sync JSON-file API; every caller now goes through EmbeddedDb.
