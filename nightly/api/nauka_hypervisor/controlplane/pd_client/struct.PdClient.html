Struct PdClient 

Source 
pub struct PdClient {
    endpoints: Vec<String>,
    timeout_secs: u64,
}
Expand description

Typed HTTP client for the PD API.

Wraps curl with retry, endpoint failover, and typed responses. Thread-safe and cheap to clone.
